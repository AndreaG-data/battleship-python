# Battleship Game in Python

## 🇬🇧 English

This project is a Python implementation of the classic Battleship game developed as part of an engineering school programming project.

The objective of this project was to strengthen programming skills in Python while working on:
- game logic implementation
- grid manipulation
- turn-based gameplay
- input validation
- error handling

Two different versions of the game were developed, progressively improving the gameplay experience and the overall program structure.

---

## 🚀 Features

- Random ship placement
- Turn-based gameplay
- Hit / miss / sunk detection
- Console-based interface
- Error handling and shot validation
- Player vs Computer mode

---

## 🛠 Technologies Used

- Python
- NumPy

# Bataille Navale en Python
## 🇫🇷 Français

Ce projet est une implémentation en Python du célèbre jeu de bataille navale réalisée dans le cadre d’un projet de programmation en école d’ingénieur.

L’objectif du projet était de développer des compétences en :

- programmation Python
- logique algorithmique
- manipulation de grilles
- gestion des erreurs
- conception de jeux interactifs

Deux versions du jeu ont été développées afin d’améliorer progressivement les fonctionnalités et l’expérience utilisateur.

---
## 🚀 Fonctionnalités

- Placement aléatoire des bateaux
- Jeu au tour par tour
- Détection des tirs touchés / manqués / coulés
- Interface console
- Gestion des erreurs et validation des tirs
- Mode Joueur vs Ordinateur

- Python
- NumPy
